# 📦 Módulo 01 — Tipagem Dinâmica e Tipos Primitivos

Este módulo explora os fundamentos da tipagem no JavaScript, com foco nos tipos primitivos e nos problemas causados pela tipagem dinâmica.

---

## 📚 Conteúdo abordado

- Tipos primitivos (`string`, `number`, `boolean`, `null`, `undefined`, `symbol`)
- Uso do `typeof` e suas limitações
- Diferença entre `==` e `===`
- Coerção de tipos
- Valores *truthy* e *falsy*
- Problemas comuns de tipagem dinâmica

## 🎯 Objetivo

- Entender como funciona a tipagem no JavaScript
- Evitar bugs causados por coerção de tipos
- Escrever funções mais seguras e previsíveis

---

## 🧠 Aprendizados

- `typeof null` retorna `"object"`
- Arrays também são considerados `"object"`
- `NaN` é do tipo `"number"`
- Comparações com `==` podem gerar resultados inesperados
- Validação de dados é essencial
