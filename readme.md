# 📦 Módulo 01 — Tipagem Dinâmica e Tipos Primitivos

Este módulo explora os fundamentos da tipagem no JavaScript, com foco nos tipos primitivos e nos problemas causados pela tipagem dinâmica.

---

## 📚 Conteúdo abordado

* Tipos primitivos (`string`, `number`, `boolean`, `null`, `undefined`, `symbol`)
* `typeof` e suas limitações
* Diferença entre `==` e `===`
* Coerção de tipos (type coercion)
* Valores *truthy* e *falsy*
* Problemas comuns causados por tipagem dinâmica

---

## 🎯 Objetivo

Desenvolver uma base sólida para:

* Identificar corretamente os tipos em JavaScript
* Evitar bugs causados por conversões automáticas
* Escrever funções mais seguras e previsíveis

## 🧠 Aprendizados

Durante este módulo, alguns pontos importantes ficaram claros:

* JavaScript pode retornar tipos inesperados (`typeof null === "object"`)
* Nem toda operação se comporta de forma intuitiva
* Validar dados é essencial em qualquer aplicação real
* Pequenos erros de tipo podem gerar bugs difíceis de encontrar
